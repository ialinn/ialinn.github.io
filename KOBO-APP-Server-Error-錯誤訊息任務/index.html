KOBO APP Server Error錯誤訊息任務

透過樂天ID以優惠價試買一本電子書，還沒閱讀前先發觸得到「Server Error in ‘/‘ Application.」的任務。

翻閱過官方指南文件進行APP故障排除的操作，完全解除不了任務便向官方提報了BUG，這回不知道是我表達文意不夠精準或者是表達錯誤，還是根本找錯NPC，到目前為止，開始起頭都是以事件來描述先讓對方能了解我遭遇到什麼困難，便以詢問事件的標題作結尾，通常客服都非常快速能給出一個解決方案。

我依然按照萬年不變的寫法，請求KOBO客服線上協助，隨機遇到攻擊力和命中數值極低的客服，首度攻擊問題核心是完全沒傷害值，我的A事件任務完全是卡關無法前進，客服不斷貼心提醒又亂點技能樹，讓人感到一股技能亂放的隨意，原因出自於我不小心沒事偷渡另一條分線任務KOBO帳號連結錯誤的詢問，此後系統信不斷回報客服又衍生出B技能及C技能，趁著技能樹未進化到D技能前，我趕緊重新檢視並整理所有的問題清楚列舉我的A事件任務原點，建議即將快要暴走的客服快回報給工程師，最起碼有更換另位客服的機會。（喂）

列舉我的錯誤：

  • 記得回報異常帳號的ID 或 電子信箱（不然會玩壞NPC）
  • 請單一詢問錯誤主事件，勿偷渡第二分線事件（一定會玩壞NPC）
  • 勿用（正常ID）去反應或突顯（異常ID）問題（可能會玩壞NPC）
  • 不要試著解釋和建議（一樣會玩壞NPC）

KOBO 網頁線上聯絡客服表單開始

因客服表單已有試過樂天ID填寫會員帳號信箱，但按下傳送完全行不通，網站回傳的小視窗寫著只接受KOBO會員註冊帳號信箱，設定像是被限制，誤以為KOBO和樂天兩大平台分開管理，還是說異常中的樂天ID沒有權利去發問，最後解決的辦法重新註冊一個帳號才有發言權限。

客服沒有再次確認帳號信箱便直接用發文者的電子信箱查詢，猜想系統應該沒顯示是KOBO或樂天，所以客服還能耐著性子解釋說明，不管使用哪個平台通通都是叫「樂天帳號」，分為新和舊帳號不能互相連結。

  • 舊會員帳號／有樂天ID：同KOBO帳號，相通各個集團直接登入。
  • 新會員帳號／沒有樂天ID：直接申請KOBO帳號，還多送樂天ID，KOBO等於樂天市場的新會員。（最划算）

綜合以上，KOBO網頁看來「我的KOBO帳戶>連結此帳戶」就顯得多餘且無用的HTML <Link>標籤，這標籤並不是記錄使用KOBO帳戶的電子信箱，讓人誤會是Facebook和Twitter的關係可以互相連結，真是沒邏輯性，結論：

我根本就是被KOBO網站和HTML <Link>標籤給誤（ㄅㄧㄠˇ）導（ㄌㄜ˙）。

誰知道第一步就踩地雷自爆，客服已說（樂天ID=KOBO）同樂天帳號相通，想先從測試KOBO連結帳號是否正常可完全放棄了。Facebook早已連結成功卻不曾使用過，沒懷疑會有異常，完全不打算浪費時間來測試與對照，但客服從頭到尾自認為這就是新舊帳號所產生錯誤息訊的問題，並朝著這方向前進。

我開始感到非常疑惑，客服也見過兩張錯誤訊息的擷圖，怎麼沒先排除比較嚴重的主線，個性使然下又才展開第二步測試社群帳號Facebook登入，連結樂天ID後，驗證下得到相同結果，畫面依舊顯示：

「抱歉，無法識別您的電子郵件帳戶。請造訪 Kobo 網站以完成註冊。」

根本沒有說好的傳送點，結果，依舊卡在KOBO網站登入的畫面，兩者非樂天集團（樂天ID≠Facebook）也非相通的帳號，為什麼還是一樣無法登入的錯誤畫面，猜想應該帳號異常或其它原因影響無法讀取才造成BUG，以往我回報經驗有類似障礙，感覺有點相關有反應給客服了解，或許可成為線索也說不定，但客服無視一直朝著新舊帳號和基本同步說明繼續兵推，增生同帳號不能合併點。

好吧！假設要合併帳號問題，我個人偏向直接按「停用帳戶」的邏輯會不會比較好。

我想求助於客服幫忙解決排除是在KOBO APP所有版本在同步和瀏覽功能這塊反常，商店首頁或同步書籍頁讀取馬上觸發錯誤訊息，明顯就是之間環節出現異常無法傳送到下一張地圖的點，並不是要客服來回應如何操作說明，討論觀點完全不一致。

總而言之，我大概被視為聽不懂人話的外星人，這個新舊帳號的點變成客服不斷跳針和鬼打牆的地方。


自己的帳號自己救、自己的任務自己解

當下重新編寫想傳遞的內容，我也Google查詢了「Server Error in ‘/‘ Application.」錯誤訊息名詞，英文單字其實有點透露出元凶，一步一步把客服拉回任務主路線。在傳送A事件前已修飾刪除那些不屬於問題的贅詞，但還是無法避免有一點情緒上的用詞，也盡可能測試來驗證結果，回覆給客服的問題並沒有偏離主題一直著墨在樂天ID異常，真希望客服能否直接快狠準的攻擊命中，只想解決這討人厭的BUG小怪。

A事件的後續，不知道是否有更換客服，突然的反應和處理比較有以往熟悉的SOP，首要幫忙先重啟帳號，後續狀況再處理，當再收到通知已經是順利破關。

「Server Error in ‘/‘ Application.」畫面隨即消失，首頁一點一點讀取，當進度條完成後立刻被傳送到全新的地圖中，展開全新探索。

這任務時間前後卡關也跑了快一個星期，不然前兩封回覆已經是夠讓人白眼翻到最高點，大概這回我忘記告知萬用的通關密語。（喂）

請客服回報給工程師處理 / 測試，謝謝 。
